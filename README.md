## Information
Minimum Blender version required (5.0.0)

## Manual Install
1. Go to [Releases](https://github.com/ThunderArtist/Red-Velvet-Theme-Blender/releases) and download the `red_velvet_theme-(version).zip` file.
2. In Blender, go to `Edit > Preferences > Themes > Install` and select the ".zip" file you downloaded in the last step.

## Custom Font
The font in the preview images is `Chakra Petch`. If you want to use it it's simple:
1. [Download Chakra Petch](https://fonts.google.com/specimen/Chakra+Petch) (Google Fonts link).
2. Extract the archive.
3. In Blender, go to `Edit > Preferences > Interface > Text Rendering > Interface Font`.
4. Select the ".ttf" file you want to use from the extracted archive.
